# Life Expectancy Analysis

This project focuses on analyzing life expectancy data across various countries and regions, with the goal of identifying factors that influence life expectancy. It includes data analysis, visualization, and exploration of key variables affecting life expectancy outcomes.

## Project Overview

- **Data Sources**: The analysis uses datasets containing life expectancy data for multiple countries. The data includes various factors such as healthcare, income, education, and other social determinants.
- **Analysis**: The project involves exploring the dataset, performing statistical analysis, and visualizing the results to identify trends and correlations.

## Files and Structure

### **STATS FINAL.xlsx**: 
Contains the main dataset used for life expectancy analysis, with different variables related to health, economy, and other factors.

### **STATS final project.docx**: 
A document detailing the methodology, analysis, and findings from the project. It provides insights into the data and conclusions drawn from the analysis.

### **Stats Project.xlsx**: 
An additional dataset or an extended version of the life expectancy data used in the analysis.
